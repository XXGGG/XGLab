## 📚 课外学习 (现阶段可不学)
*这些知识在 Nuxt 开发中可能暂时用不到，或者有更好的替代方案，先放一边！* 🗑️

- `v-model` 的自定义组件用法 (Nuxt 表单库通常封装好了)
- 自定义指令 `directives` (高级用法)
- `provide` / `inject` (初学者容易乱，先用 Props)
- Vuex / Pinia (状态管理，Nuxt 有 `useState`，后面再学 Pinia)
- Options API (我们只学 Composition API，因为 Nuxt 推荐！)
