# 03-点它点它 👆

## 🎮 像玩游戏一样互动

网页如果只能看不能点，那跟报纸有什么区别？
今天我们来学习怎么监听用户的动作，比如**点击**。

## 1. 事件监听 `v-on` (或者叫 `@`)

就像 `v-bind` 的缩写是 `:` 一样，`v-on` 也有个超酷的缩写：`@`。
`@click` 就代表：“当被点击的时候...”。

## 2. 定义一个动作 (函数)

在 `<script setup>` 里，我们可以定义一个函数（Function），也就是一个动作。

### 🌰 举个栗子

```html
<script setup>
  // 定义一个动作：打招呼
  const sayHello = () => {
    alert("你好呀！👋");
  };

  // 定义一个动作：甚至可以接收参数
  const eat = (food) => {
    alert("我正在吃：" + food);
  };
</script>

<template>
  <!-- 当按钮被点击时，执行 sayHello 这个动作 -->
  <button @click="sayHello">点我打招呼</button>

  <!-- 传递参数 -->
  <button @click="eat('汉堡包')">吃汉堡</button>
  <button @click="eat('披萨')">吃披萨</button>
</template>
```

## 🤓 小知识：什么是 `() => {}` ?

这叫**箭头函数**。
- `const sayHello = () => { ... }`
- 翻译成人话就是：创建一个叫 `sayHello` 的东西，它是一个动作，动作的内容在 `{ ... }` 里面。

## 🎒 课后小结

- 想监听点击？用 `@click`。
- 想监听输入？用 `@input`。
- 想监听回车？用 `@keydown.enter` (Vue 很贴心吧！)。
- 把动作写在 `<script setup>` 里，然后在 `<template>` 里用 `@` 呼唤它。
