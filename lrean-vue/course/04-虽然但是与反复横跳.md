# 04-虽然但是与反复横跳 🤹

## 🤖 机器人也要有逻辑

生活不总是直线的，有时候我们要面临选择（如果...就...），有时候我们要重复做一件事（把这些砖搬完...）。
Vue 也能处理这些逻辑！

## 1. 条件渲染 `v-if` (虽然但是)

`v-if` 就像一个开关。如果条件是真的，它就显示；如果是假的，它就消失（从 DOM 中完全移除，连渣都不剩）。

### 🌰 举个栗子

```html
<script setup>
  const isHappy = true; // 开心吗？
  const score = 59; // 分数
</script>

<template>
  <!-- 如果 isHappy 是 true，这行字才会出现 -->
  <h1 v-if="isHappy">😄 我今天很开心！</h1>
  <!-- 否则（v-else），显示这个 -->
  <h1 v-else>😭 我想哭...</h1>

  <!-- 多重判断 -->
  <div v-if="score >= 90">优秀！🌟</div>
  <div v-else-if="score >= 60">及格了！👌</div>
  <div v-else>回家准备挨打吧...👋</div>
</template>
```

> **小贴士**：还有一个叫 `v-show` 的家伙。它和 `v-if` 很像，但 `v-show` 只是用 CSS (`display: none`) 把元素藏起来，元素还在那里。如果你要频繁切换显示/隐藏，用 `v-show` 性能更好哦！

## 2. 列表渲染 `v-for` (影分身之术)

如果你有一堆数据要展示，不用写一堆 `div`，用 `v-for` 循环出来！

### 🌰 举个栗子

```html
<script setup>
  // 一个数组（Array），里面装着水果
  const fruits = ['苹果', '香蕉', '橘子', '榴莲'];
  
  // 一个数组对象
  const users = [
    { id: 1, name: '小明' },
    { id: 2, name: '小红' }
  ];
</script>

<template>
  <ul>
    <!-- 语法：item in list -->
    <!-- 意思：从 fruits 里拿出一个，叫 fruit，然后把 li 重复好多次 -->
    <!-- :key 是身份证，必须要有！Vue 靠它认人，通常用唯一的 id 或者索引 -->
    <li v-for="(fruit, index) in fruits" :key="index">
      第 {{ index + 1 }} 种水果是：{{ fruit }}
    </li>
  </ul>

  <div>
    <!-- 循环对象数组 -->
    <div v-for="user in users" :key="user.id">
      用户ID: {{ user.id }} - 姓名: {{ user.name }}
    </div>
  </div>
</template>
```

## 🎒 课后小结

- 想控制显隐？用 `v-if` 或 `v-show`。
- 想循环展示？用 `v-for`，记得加上 `:key`（这点很重要，不加 Vue 会发脾气警告你）。
