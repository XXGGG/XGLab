# 08-乐高积木 🧱

## 🏙️ 别把鸡蛋放在一个篮子里

如果你的网页很大，有几千行代码。
如果全写在一个 `.vue` 文件里，那简直是灾难！找个变量都要翻半天。

Vue 允许我们把页面拆分成一个个小的 **组件 (Component)**。
就像搭乐高：
- 头部组件 (Header.vue)
- 侧边栏组件 (Sidebar.vue)
- 内容组件 (Content.vue)

然后把它们拼在一起！

## 1. 定义组件

其实任何一个 `.vue` 文件都可以是一个组件。
假设我们建一个 `MyButton.vue`：

```html
<!-- MyButton.vue -->
<template>
  <button class="cool-btn">我是超级按钮</button>
</template>

<style>
  .cool-btn { background: pink; border: none; padding: 10px; }
</style>
```

## 2. 引入和使用组件

在父组件（比如 `App.vue`）里使用它：

```html
<!-- App.vue -->
<script setup>
  // 1. 引入组件
  import MyButton from './MyButton.vue';
  
  // 2. 在 setup 语法糖里，引入了就能直接用！不需要注册！
</script>

<template>
  <div>
    <h1>我的主页</h1>
    <!-- 3. 像写 HTML 标签一样使用它 -->
    <MyButton />
    <MyButton />
    <MyButton />
  </div>
</template>
```

你看，我们复用了三次按钮，但只写了一次代码！这就是组件化的魅力。

## 🚀 Nuxt 小预告

在 Vue 中你需要手动 `import`。
但是！等你以后学了 **Nuxt**，你会发现 Nuxt 会自动帮你引入组件（Auto Imports）。你连 `import` 都不用写，直接在 template 里写 `<MyButton />` 就能用！
是不是很爽？不过现在我们先老老实实写 import，打好基础。

## 🎒 课后小结

- 拆分代码，把独立的功能写成 `.vue` 文件。
- `import` 进来，直接当标签用。
- 组件名通常用 **大驼峰** (PascalCase)，比如 `MyButton`。
