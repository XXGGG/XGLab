# 02-把数据秀出来 📺

## 🗣️ 别光想，说出来！

上一节课我们知道了 Vue 的大脑（Script）和身体（Template）。
现在，大脑里有个想法，怎么让身体把它表现出来呢？

## 1. 插值表达式 `{{ }}`

这玩意儿长得像两撇小胡子，所以也叫 Mustache 语法。
它的作用就是：**把 JS 里的变量，挖个坑填到 HTML 里。**

### 🌰 举个栗子

```html
<script setup>
  const name = "奥特曼";
  const power = 9999;
</script>

<template>
  <!-- 页面上会显示：我是奥特曼 -->
  <h1>我是 {{ name }}</h1>
  
  <!-- 页面上会显示：战斗力：9999 -->
  <p>战斗力：{{ power }}</p>
  
  <!-- 甚至可以做简单的计算！显示：10000 -->
  <p>升级后：{{ power + 1 }}</p>
</template>
```

## 2. 绑定属性 `v-bind` (或者叫 `:`)

有时候，我们想改变 HTML 标签的**属性**，比如图片的 `src`，或者链接的 `href`。
这时候不能用 `{{ }}` 哦！要用 Vue 的特技：`v-bind`。

因为 `v-bind` 太长了，我们通常只写一个冒号 `:`。这是它的缩写（你可以理解为昵称）。

### 🌰 举个栗子

```html
<script setup>
  const myAvatar = "https://example.com/face.png";
  const myLink = "https://www.bilibili.com";
</script>

<template>
  <!-- ❌ 错误写法：不能在属性里用胡子 -->
  <!-- <img src="{{ myAvatar }}"> -->

  <!-- ✅ 正确写法：用冒号！ -->
  <!-- 意思是：把 src 这个属性，绑定到 myAvatar 变量上 -->
  <img :src="myAvatar" />

  <!-- 意思是：把 href 绑定到 myLink -->
  <a :href="myLink">点我去B站</a>
</template>
```

## 🎒 课后小结

- 想把文字显示在标签**中间**？用双大括号 `{{ }}`。
- 想把变量给标签的**属性**（如 src, href, class）？用冒号 `:`。
