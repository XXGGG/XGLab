# 03-说话要优雅.md

🗣️ 欢迎来到第三课！

你是不是经常写出这种代码：`"Hello, " + name + "! You have " + count + " messages."`？
写完是不是觉得自己像个拼图工，到处找 `+` 号和引号？😫

ES6 带来了一个神器：**模板字符串 (Template Literals)**。从此，拼串变得极其优雅。

## 🐣 基础用法

### 1. 换个符号

以前我们要用单引号 `'` 或双引号 `"`。
现在，请找到你键盘左上角（Esc 下面）的那个键，它叫**反引号** `` ` ``。

```javascript
const str = `我是模板字符串`;
```

### 2. 随意插入变量

在反引号里，用 `${}` 包裹变量，想插哪里插哪里！

```javascript
const name = "Trae";
const action = "写代码";

// 以前
const msgOld = name + "正在" + action + "，请勿打扰。";

// 现在
const msgNew = `${name}正在${action}，请勿打扰。`; 
// 是不是清爽多了？
```

### 3. 支持换行

以前字符串换行要写 `\n` 或者用 `+` 号拼接，很丑。
模板字符串里，直接回车就行！

```javascript
const menu = `
  今天吃什么：
  1. 宫保鸡丁
  2. 鱼香肉丝
  3. 西红柿炒蛋
`;
console.log(menu);
```

## 🎭 场景实战

### Vue (Nuxt) 中的用法

虽然 Vue 模板里直接写 HTML，但在 `<script>` 里处理逻辑或者生成动态类名时很有用。

```javascript
<script setup>
const id = 123;
// API 请求路径拼接
const url = `/api/user/${id}/details`; 

// 动态生成样式字符串（虽然通常建议用 :style 对象语法）
const color = 'red';
const styleStr = `color: ${color}; font-size: 20px;`;
</script>
```

### React (Next) 中的用法

在 React 中，处理 className 时经常用到。

```javascript
const Button = ({ isActive, label }) => {
  // 动态拼接 class
  // 注意：实际项目中通常会用 'classnames' 或 'clsx' 库，但原理一样
  const className = `btn ${isActive ? 'btn-active' : ''} p-4`;

  return (
    <button className={className}>
      {`点击我: ${label}`}
    </button>
  );
};
```

## 🧠 总结

- 只要字符串里需要包含变量，**立马使用反引号 `` ` ``**。
- 只要字符串需要换行，**立马使用反引号 `` ` ``**。
- `${}` 里面其实可以写任何 JavaScript 表达式（比如运算、函数调用），但建议保持简单，别写太复杂的逻辑。
