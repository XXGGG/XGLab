## 🏗️ 任务：超级待办事项 (Super Todo List)

我们要用 Vite 搭建一个项目，实现一个带有 TS 类型的 Todo List。

### 1. 初始化项目

请打开终端，选择你喜欢的框架运行命令：

**Vue 选手：**
```bash
pnpm create vite my-vue-ts-app --template vue-ts
```

**React 选手：**
```bash
pnpm create vite my-react-ts-app --template react-ts
```

然后进入目录，安装依赖：
```bash
cd my-xxx-app
pnpm install
pnpm dev
```

### 2. 功能要求

你的应用需要实现以下功能，并且**必须使用 TypeScript**：

1.  **数据结构**：定义一个 `Todo` 接口。
    ```typescript
    interface Todo {
      id: number;
      text: string;
      done: boolean;
    }
    ```
2.  **列表展示**：使用一个数组状态（`ref<Todo[]>` 或 `useState<Todo[]>`）来存储数据，并循环展示在页面上。
3.  **添加功能**：有一个输入框和按钮。输入文字，点击按钮，向数组添加一项（ID 可以用 `Date.now()` 生成）。
    - *注意：输入框绑定的变量应该是 string 类型。*
4.  **删除功能**：每项后面有个“删除”按钮，点击后根据 ID 删除该项。
    - *注意：删除函数的参数应该是 id: number。*
5.  **切换状态**：点击文字，可以在“已完成/未完成”之间切换（修改 `done` 属性）。

### 3. 严格模式挑战 (Hard Mode)

如果你觉得上面的太简单，试试这个：
-   给 `Todo` 接口加一个可选属性 `tag?: 'work' | 'life'` (字面量联合类型)。
-   添加时，允许用户选择标签。
-   展示时，根据标签显示不同颜色。

1.  **看红线**：TS 最强大的地方就是报错。鼠标移上去，看看它说了什么。
2.  **搜一搜**：把报错信息复制到 Google/ChatGPT，通常都能找到答案。
3.  **回看教程**：忘记了怎么定义数组？忘记了怎么写 Props？回去翻翻前面的笔记。
