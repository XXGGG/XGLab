# TypeScript 小白速成班 - 课程安排 🎓

嗨！欢迎来到 TypeScript 的世界！我是你的讲师。
别担心，这里没有枯燥的理论，只有实用的干货。我们的目标是：**能在 Nuxt 和 Next 中愉快地写代码，而不是成为语言学家。** 🚀

我们将由浅入深，一步步征服 TS。每节课都包含基础知识 + Vue/React 场景应用 + 课后小练习。

## 📅 课程表

### 第一阶段：打好地基 (基础篇)
这些是你在任何 TS 项目中都会用到的基础知识，必须掌握哦！

- **[01-环境搭建与初体验](./01-环境搭建与初体验.md)**
    - 内容：安装 pnpm, tsx, 运行你的第一个 `.ts` 文件。
    - 目标：跑通流程，不再害怕黑底白字的报错。
- **[02-基础类型大阅兵](./02-基础类型大阅兵.md)**
    - 内容：`string`, `number`, `boolean`, `any` (以及为什么别乱用它)。
    - 目标：学会给变量“贴标签”。
- **[03-数组与元组](./03-数组与元组.md)**
    - 内容：怎么定义一串数据，元组(`Tuple`)又是个什么怪东西。
    - 目标：管好你的列表数据。
- **[04-对象与接口](./04-对象与接口.md)**
    - 内容：`interface` 的使用，如何描述一个复杂的对象。
    - 目标：给你的 API 数据穿上“制服”。
- **[05-函数的小秘密](./05-函数的小秘密.md)**
    - 内容：给参数查票，给返回值做预告。
    - 目标：写出让队友敢放心调用的函数。

### 第二阶段：进阶一点点 (常用篇)
掌握这些，你就能读懂 80% 的业务代码了。

- **[06-联合类型与类型别名](./06-联合类型与类型别名.md)**
    - 内容：`type` vs `interface`，`|` (或) 操作符。
    - 目标：处理“既可能是这个，又可能是那个”的情况。
- **[07-泛型是什么鬼](./07-泛型是什么鬼.md)**
    - 内容：泛型(`Generic`)的通俗解释，简单的泛型函数。
    - 目标：看懂 `<T>` 是什么意思，写出更通用的代码。

### 第三阶段：实战演练 (框架篇)
结合你熟悉的 Vue 和 React，看看 TS 到底怎么用。

- **[08-TS在Vue中的应用](./08-TS在Vue中的应用.md)**
    - 内容：`defineProps`, `ref`, `reactive` 的 TS 写法。
    - 目标：在 Nuxt/Vue3 中优雅地使用 TS。
- **[09-TS在React中的应用](./09-TS在React中的应用.md)**
    - 内容：组件 Props, `useState`, 事件处理函数的类型。
    - 目标：在 Next/React 中不再满屏飘红。

### 第四阶段：毕业设计
- **[10-实战作业：搭建你的第一个TS项目](./10-实战作业：搭建你的第一个TS项目.md)**
    - 内容：使用 Vite 搭建环境，完成一个小的 TodoList。
    - 目标：综合运用所学知识，从头开始！

---

## 📚 课外学习 (现阶段可以不看)
这些知识在日常业务开发中用得少，或者是为了写库准备的，小白可以先跳过：
- 复杂的泛型约束与推断
- `never`, `unknown` 的深层区别
- 类型体操 (Conditional Types, Mapped Types 等)
- 装饰器 (Decorators)
- 命名空间 (Namespaces - 现在都用 Module 了)
