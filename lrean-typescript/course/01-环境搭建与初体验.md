# 01-环境搭建与初体验 🛠️

## 👋 嗨！
欢迎来到第一节课！我知道你可能已经迫不及待想写代码了，但在那之前，我们需要先准备好工具。这就好比大厨做饭前得先磨刀一样。🔪

别担心，这节课非常简单，我们主要的目标是：**让你的电脑能跑懂 TypeScript 代码**。

---

## 📦 1. 准备工具

我们需要用到以下神器：
- **Node.js**: 前端运行的基础环境（我相信你肯定有了）。
- **pnpm**: 一个又快又省磁盘空间的包管理工具（比 npm/yarn 香多了）。
- **tsx**: 一个能直接运行 TS 文件的黑科技，学习阶段用它简直不要太爽！不用配置繁琐的 `tsconfig.json` 就能跑起来。

### 安装 pnpm (如果你还没有)
打开你的终端（Terminal），输入：
```bash
npm install -g pnpm
```
*💡 小贴士：如果安装报错，可能需要管理员权限 (Windows) 或 sudo (Mac/Linux)。*

### 初始化学习项目
找个风水宝地（比如 `lrean-typescript` 文件夹），我们初始化一个项目：
```bash
pnpm init
```
这会生成一个 `package.json` 文件，它是项目的身份证。

### 安装 tsx
我们把 `tsx` 安装为开发依赖：
```bash
pnpm add -D tsx
```

---

## ✍️ 2. 写下你的第一行 TS

在项目根目录下，新建一个文件叫 `01-hello.ts`。
注意后缀是 `.ts` 哦！这就是 TypeScript 的标志。

输入以下代码：

```typescript
// 01-hello.ts

// 这是一个普通的字符串变量
const greeting: string = "Hello, TypeScript World! 🌍";

// 打印它
console.log(greeting);

// 顺带讲个 ES6 小知识 💡：
// const 用来定义常量（不能变的），let 用来定义变量（能变的）。
// 尽量多用 const，除非你确定这个值后面要改。
```

### 🧐 仔细看
发现了吗？我们在 `greeting` 后面加了个 `: string`。
这就是 TypeScript 的核心——**类型注解**。
我们在告诉电脑：“`greeting` 这个变量，必须是字符串，敢塞数字进来我就报错！”

---

## 🚀 3. 运行它！

以前我们运行 JS 用 `node xxx.js`，现在我们用 `tsx`：

```bash
npx tsx 01-hello.ts
```
*(或者如果你配置了 script，也可以用 pnpm run xxx，但现在直接用 npx 最快)*

如果你在终端看到了 `Hello, TypeScript World! 🌍`，恭喜你！🎉 你已经成功迈出了第一步！

## 🖼️ Vue 和 React 场景

虽然这节课是环境搭建，但也让你看看 TS 在框架里长啥样，混个眼熟：

### Vue (Script Setup)
```vue
<script setup lang="ts">
// 注意那个 lang="ts"
const title: string = "我是 Vue 标题";
</script>
```

### React (JSX)
```tsx
// 组件文件通常叫 .tsx
const Title = () => {
  const title: string = "我是 React 标题";
  return <h1>{title}</h1>;
};
```
*看到没？用法几乎一模一样！学会了 TS，走遍天下都不怕。*
