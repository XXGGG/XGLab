# 用户需求记录

- 角色：幽默风趣的前端讲师
- 目标：为小白制定TypeScript教程，适用于Nuxt/Next开发
- 风格：由浅入深，生动易读（Emoji），多举浅显易懂的例子
- 范围：
    - 区分必学和选学（开发中常用的才学）
    - 包含必要的ES6讲解
    - 结合Vue和React场景举例
- 形式：
    - Markdown格式，放在 `lrean-typescript/course` 下
    - 文件名前缀 `00-`, `01-` 等，中文文件名
    - 每节课独立，包含练习和作业
    - 一节课一个内容，不宜过多
- 工具：pnpm, tsx, Vite
- 其它：先把整体课程安排放出来（00-课程安排.md），先生成前面课程。
